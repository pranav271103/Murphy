# Murphy: Project Identity

## What This Is
Murphy is a high-speed agency coding platform designed for autonomous development. It uses a dual-model orchestration approach to balance speed and intelligence, focused on CLI-first interactions.

## Core Value
To provide a predator-like, high-precision coding environment where agents can execute complex tasks with minimal human intervention.
